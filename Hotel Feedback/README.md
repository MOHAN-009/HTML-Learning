# Hotel Feedback Page

This project is a simple hotel feedback page built using HTML and CSS.

## Features

- User feedback form
- Responsive design
- Easy to customize

## Usage

1. Open `index.html` in your browser
2. Fill out the feedback form
3. Submit your feedback
